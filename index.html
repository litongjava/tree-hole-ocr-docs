Tree Hole OCR

Open source free text recognition tool

Get Started

Local OCR Recognition

Treehole OCR Text Recognition tool operates without the need for an internet connection, utilizing local OCR technology based on the Paddle OCR model and deep learning frameworks such as PyTorch and DJL, providing fast and accurate text recognition.

Cross-Platform Compatibility

Developed using Java 1.8 and JavaFX, it supports operation on various operating systems, including Mac OS X 12.6 and higher.

Powerful Feature Support

In addition to basic text recognition, it includes PDF recognition, image text recognition, and quick-access screenshot recognition.
